#  Weather Information System

A full-stack **Weather Information System** that provides real-time weather forecasts, astronomy data, and AI-generated outfit suggestions.  
Built with **FastAPI** for the backend and **React.js** for the frontend, it integrates **WeatherAPI** for live weather data and **OpenAI GPT** for smart, natural language suggestions.

---

##  Features

-  **Live Weather Data** — Fetches temperature, humidity, wind, precipitation, and more.
-  **7-Day Forecast** — Displays upcoming weather conditions visually.
-  **Astronomy Info** — Sunrise, sunset, and moon phase details.
-  **AI Outfit Suggestions** — Uses OpenAI GPT to suggest what to wear based on current weather.
-  **Location-Based Search** — Search by city name or coordinates.
-  **Favorites / Saved Cities** — Quickly revisit your favorite places.
-  **Responsive UI** — Built with React for a clean, interactive design.
